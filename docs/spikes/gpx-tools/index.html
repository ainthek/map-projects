Gpx Tools

Experimental collection of tools (v.0.0.2) Implemented in JS. Drag file to first textarea.

All-in-one

performs all below in sequence

Remove zero speed points

Each point with zero speed is deleted

Detect stops

Detects "autostops" in gpx files. Creates wpt for each trackpint where deltaT>60s.

Add/Process Extensions

Removes all extensions and replaces them with calculated extension in flat format to be usable by QGis import (hates nested extensions)

segmentize

creates segment (line) for each 2 points
